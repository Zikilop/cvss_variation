# CVSS Scenario Generator

Программный комплекс для автоматической генерации сценариев клонирования моделей
сердечно-сосудистой системы в формате, совместимом с солвером CVSS (версия 2.5+).

## Назначение

При моделировании гемодинамики часто требуется создать большое количество вариантов
одной модели с различными параметрами сосудов (сечения, давления, длины и т.д.).
Ручное создание таких сценариев через GUI нецелесообразно при тысячах вариантов.

Данный инструмент позволяет:
- описать пространство варьируемых параметров в компактном YAML-файле
- автоматически сгенерировать текстовый сценарий клонирования (`.txt`)
- использовать сетку параметров или случайное сэмплирование
